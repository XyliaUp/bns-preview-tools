功能
==========================
如需获得更多信息，请查看提交日志
## 物品
获取列表   
查看指定物品
在搜索界面中查看物品信息

## 任务
获取列表
获取主线任务列表 
查看指定任务

## 文本
比较不同版本间的更新
输出文件

## 资源文件
本软件并非用于查看虚幻引擎资产, 只提供输出文件的功能。
如需更多特性, 请使用 [FModel](https://github.com/4sval/FModel)

## 角色属性计算
此功能存在一些瑕疵   

环境
==========================
.NET 8.0 SDK    
Windows 7 更高
